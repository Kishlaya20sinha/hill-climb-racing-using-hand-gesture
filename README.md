This project uses hand gestures (via webcam) to simulate key presses for controlling games like Hill Climb Racing.

## Technologies Used
- Python 🐍
- OpenCV 🎥
- MediaPipe ✋
- pynput ⌨️

## How It Works
- Raise 3+ fingers to **Accelerate** (Right arrow key).
- Raise fewer fingers to **Brake** (Left arrow key).
